MERN STACK

The MERN stack is a popular collection of JavaScript-based technologies used to build dynamic web applications. It stands for MongoDB (database), Express.js (backend framework), React (frontend library), and Node.js (runtime environment). This stack allows developers to use a single programming language across the entire application

The Four Components of MERN Stack

  • MongoDB:A NoSQL document database used to store application data in a flexible, JSON-like format.(1st)
  • Express.js:A back-end web framework used to build API'S and handle HTTP requests.(2nd)
